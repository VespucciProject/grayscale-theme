Download Vespucci


Vespucci binary packages can be found on the GitHub Releases page

Vespucci is distributed as a portable package. There is nothing to "install". On Windows, run bin/vespucci.exe. On Mac, right-click the .app file and select open. On linux run vespucci.sh from the root of the file.

System Requirements

  • Windows: Windows 7+, 64-bit
  • macOS: macOS 10.7 (Lion) or later
  • Linux: Ubuntu 14.04+ or other distros of similar vintage

On Windows, the Microsoft Visual C++ 2015 runtime is required. To install it, run vcredist_x64.exe. It is quite possible that you already have this installed.

On Linux, you must have BLAS/LAPACK (which ship with nearly every distro), HDF5 and Qt 5.5+. We do not guarantee that this will work on every distribution. You may wish to compile Vespucci yourself. Look at the .travis.yml file to see how we do this on Travis CI
