Documentation and Help

The Vespucci user manual is currently being updated to the new UI an older PDF version of the manual is availible here. HTML documentation is published to the gh-pages branch of the VespucciProject/Vespucci-docs repository and can be viewed at vespucciproject.org/Vespucci-docs C++ API documentation can be viewed at vespucciproject.org/Vespucci-docs/doxygen

To report bugs, please use the GitHub Issues Page or contact by email.

If you would like to contribute to Vespucci, please puruse CONTRIBUTING.md or contact us via GitHub or email.
